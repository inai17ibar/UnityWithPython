# UnityWithPython
 
# Features
 
Windows10で、UnityのC#からPythonを実行する方法をプロトタイプする

 
# Requirement
 
Unityは動作確認済みのバージョン
 
* Unity 2020.3.7f1
* Python 3.6
 
# Installation
 
* Unityのインストール
* Python3.6のインストール

# Usage
 
```bash
git clone https://github.com/hoge/~
cd examples

# UnityEditorでプロジェクトを開く
```
 
PythonExetutorのPythonのPathを自分のPC環境と一致させ, Editorから実行する
